        1.图片预览区域
  <ul class="picList" modulecd="/test/"></ul>  
   modulecd   : 必填项
   moduleno   : 选填项, 如果有moduleno,则为 <ul class="picList" modulecd="/test/" moduleno="moduleno"></ul>  
   extensions : 选填项, 限制上传文件类型，默认值"jpg,jpeg,png,gif,bmp,tiff,raw,zip", 如需自定义<ul class="picList" modulecd="/test/" extensions="png,jpg"></ul> 
   size       : 选填项, 限制上传文件大小，默认值"500kb", 如需自定义<ul class="picList" modulecd="/test/" size="*kb"></ul> 
   multi      : 选填项, true:可选择多个文件,false:只能选择一个文件，默认值为true，如需自定义 <ul class="picList" modulecd="/test/" multi="false"></ul> 
   url        : 选填项, 如需自定义 <ul class="picList" modulecd="/test/" url="/v1/api/common/upload-files.form"></ul> 
   params     : 选填项, 如需自定义 <ul class="picList" modulecd="/test/" params="{'param1':'val1','param2':'val2'}"></ul> 
   headtitle  : 选填项, 如需自定义 <ul class="picList" modulecd="/test/" headtitle="title"></ul> 
   addtext    : 选填项, 如需自定义 <ul class="picList" modulecd="/test/" addtext="addtext"></ul> 
   starttext  : 选填项, 如需自定义 <ul class="picList" modulecd="/test/" starttext="starttext"></ul>
